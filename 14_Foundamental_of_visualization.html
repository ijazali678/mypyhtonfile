Data visualization and fundamental of data visualization¶

Graphic system Plots convert code into graph that is called data visualization. the most important part of data science is to represent the data into graph

what is the grammer of graphics?¶

Major Parts

1- Mapping (Data)

2- Aestheric (Color, Shape, Size)

3- Geometric (Ojects: Line, Bar, Points, Box, Map)

visualization depends on the variable type

1- Categorical variable

-Counts

think before you start¶

  • generate a hypothesis (e.g how much male and female in course)

  • find a story in your data

  • clean/tidy up data

  • remove duplicates

  • handle missing values

  • find a proper variable to drop duplicates in data
